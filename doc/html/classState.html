Description détaillée

Classe représentant un état de l'application.

Un State est composé d'une initialisation, une boucle, un affichage (texte et graphique), destructeur... State est une classe virtuelle pure : elle doit être héritée et toutes les fonctions héritée par State implémentée. Exemple de State : SplashScreen, MainMenu, Game.
