Description détaillée

Etat de "balade" d'un Animal, il se déplace aléatoirement.

L'Animal se balade sur la Map en prenant en compte les collisions et change de direction aléatoirement. Il entre dans l'état AnimalFollowing si le joueur est à proximité.
