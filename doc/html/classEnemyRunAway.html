Description détaillée

Etat de fuite pour les enemy, hérite de StateBehavior<Enemy>.

L'Enemy se déplace dans la direction opposé du joueur, si il est assez loin du joueur il entre en état EnemyPatrol.

Paramètres

[in]player_: std::unique_ptr<Player>, le joueur.
[in]collision: pointeur sur CollisionLayer, permet d'avoir accès à la couche de collision pour les déplacements.
